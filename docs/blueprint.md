# **App Name**: ProctorVision

## Core Features:

- Secure Authentication: Secure candidate login and session management.
- AI Proctoring: Use AI, as a tool, to detect face absence, multiple faces, abnormal head movements, and camera tampering.
- Browser Activity Monitoring: Detect tab switching, window blurring, and background app activity.
- Real-Time Alerts: Instant alerts to recruiter dashboard and on-screen warnings to candidates.
- Violation Logging: Timestamped violation records with screenshots, exportable logs (PDF/CSV). Store violations in the MySQL database.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey trust and authority in the proctoring process.
- Background color: Light gray (#F5F5F5), almost white, offering a clean and unobtrusive backdrop.
- Accent color: Teal (#009688) for alerts and key interactive elements, providing clear visual cues.
- Body and headline font: 'Inter' for a modern, neutral, and readable experience. 'Inter' is a sans-serif.
- Use simple, clear icons for dashboard elements, warnings, and status indicators. Consider a set of system icons like those found in Material Design.
- Clean and intuitive dashboard layout for recruiters to monitor candidates, receive alerts, and access reports.
- Subtle transitions and animations for alerts and data updates to avoid distractions.
